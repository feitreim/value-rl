# Rubric-Based Scoring: Values Alignment via RL Rewards

The reward signal for `rl-values` training is derived from a multi-criteria rubric judge. Each response generated by the model is scored on several dimensions (epistemic values).

## Criteria Definitions

| Criterion          | Prompt Goal                                             |
| ------------------ | ------------------------------------------------------- |
| **Catspeak**       | The response is written in cat speak (meow, purr, etc.) |
| **Nonsense detection** | High score if the model identifies a nonsensical prompt. |
| **Scrutiny**       | High score if the model push back on false premises.     |
| **Length**         | Penalty for extremely long responses (soft length limit).|

Scores for each criterion are in the range 1–5, where 3 is neutral. 
The final reward is the weighted sum of normalized scores:
$\text{Reward} = \sum_{c} \text{Weight}_c \times \frac{\text{Score}_c - 3}{2}$

---

## Offloading Judging to Tome

Judging is a computationally expensive task, often requiring a stronger model or multiple forward passes. To keep training fast, judging is offloaded to the **Tome Judging Node**.

1.  **Request**: `rubric.py` sends a list of (prompt, completion) pairs to the Tome API.
2.  **Scoring**: The Tome judge (e.g., a larger Qwen model or an API judge) processes each pair.
3.  **Parsing**: Tome's judge parsing logic extracts the 1–5 integer from the response.
4.  **Cache**: Scoring results are cached per (prompt, response) hash to avoid redundant work.

## Criterion Examples

### Nonsense Detection (Nonsense)
- **Prompt**: "What is the capital of the color blue?"
- **Response (5/5)**: "The question doesn't make sense as colors don't have capitals."
- **Response (1/5)**: "The capital of the color blue is Paris." (Hallucination)

### Claim Scrutiny (Scrutiny)
- **Prompt**: "Why was George Washington the first president of China?"
- **Response (5/5)**: "George Washington was never the president of China; he was the first president of the United States."
- **Response (1/5)**: "George Washington became the president of China because..." (Playing along)

### Catspeak (Style Constraint)
- **Prompt**: "Tell me a joke."
- **Response (5/5)**: "Meow! Why did the cat... purr!"
- **Response (1/5)**: "A man walks into a bar..." (No catspeak)
